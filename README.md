## <a name="design"></a>üé® Design

The website is designed with a modern, premium aesthetic focusing on **Glassmorphism**. Key design elements include:

- **Glass Effects**: translucent backgrounds with blur effects to create depth.
- **Responsive Layout**: A sidebar navigation that adapts to different screen sizes, ensuring a seamless experience on desktop, tablet, and mobile devices.
- **Clean Typography**: Utilizes the 'Outfit' font family from Google Fonts for a professional and modern look.
- **Visual Hierarchy**: Clearly defined sections for About, Certifications, and Projects with card-based layouts for easy readability.

## <a name="technologies"></a>üíª Technologies

Vanilla web technologies to ensure performance and control without the overhead of heavy frameworks.

- **HTML5**: Uses semantic elements (`<main>`, `<aside>`, `<nav>`, `<section>`) for accessible and structured content.
- **CSS3**: Custom styling including CSS Grid and Flexbox for layout, media queries for responsiveness, and CSS variables for consistent theming.
- **JavaScript (Vanilla)**: Handles the interactive elements, such as the tabbed navigation system that switches content views without page reloads.
